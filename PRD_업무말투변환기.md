## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 사용자가 입력한 일상적인 비즈니스 의사소통 내용을 AI(Solar-Pro2)를 활용하여 수신 대상(상사, 동료, 고객 등)에 최적화된 정중하고 전문적인 업무 말투로 변환해 주는 웹 서비스입니다.

## 2. 목표 및 핵심 가치 (Goals & Core Value)
- **목표**: 비즈니스 메일이나 메시지 작성 시 소요되는 고민 시간을 단축하고 커뮤니케이션의 전문성을 높임.
- **핵심 가치**: "핵심은 유지하되, 표현은 격식 있게" - 원문의 의도를 훼손하지 않고 상황에 맞는 적절한 문체로 변환.

## 3. 사용자 및 유즈케이스 (User & Use Cases)
- **신입 사원**: 상사에게 어려운 보고나 일정 조율을 요청할 때 적절한 어휘 선택이 고민되는 상황.
- **실무자**: 타 부서나 고객에게 수정 사항을 요청하거나 사과 메시지를 보낼 때 정중함을 유지해야 하는 상황.

## 4. 기능적 요구사항 (Functional Requirements)

### 4.1 핵심 기능: 말투 변환
- **F-01: 입력 기능** - 사용자는 변환하고자 하는 핵심 내용을 텍스트 형태로 자유롭게 입력할 수 있어야 함.
- **F-02: 대상 선택** - 사용자는 다음 4가지 수신 대상을 선택할 수 있어야 함.
    - 👔 **상사 / 임원**: 격식 있고 공손한 경어체 (극존칭 포함)
    - 🤝 **타팀 동료**: 정중하면서도 협조적인 비즈니스 어조
    - 🌐 **고객 / 외부**: 친절하고 신뢰감을 주는 서비스 어조 (~입니다/합니다 체)
    - 👥 **팀 내 동료**: 간결하고 효율적인 실무 어조
- **F-03: AI 변환** - 선택된 대상과 입력된 텍스트를 바탕으로 AI가 최적화된 메시지를 생성함.
- **F-04: 결과 출력** - 변환된 결과물을 화면에 표시하고, 사용자가 쉽게 복사할 수 있는 기능을 제공함.

### 4.2 제외 범위 (Out of Scope)
- 사용자 인증 (로그인/회원가입)
- 변환 내역 저장 및 관리 데이터베이스
- 다국어 지원 (한국어 전용)

## 5. 비기능적 요구사항 (Non-functional Requirements)
- **성능**: AI 요청 후 3~5초 이내에 결과가 응답되어야 함.
- **사용성**: 모바일 및 데스크톱 브라우저에서 모두 깨지지 않는 반응형 웹 디자인 적용.
- **보안**: API Key가 클라이언트(프론트엔드) 코드에 노출되지 않도록 백엔드 서버에서 처리함.

## 6. 기술 스택 (Tech Stack)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Python 3.10+, FastAPI
- **AI Engine**: Upstage Solar-Pro2 (via LangChain)
- **Deployment**: Vercel

## 7. 화면 설계안 (UI Design)

### 7.1 메인 레이아웃
- **Header**: 서비스 로고 및 타이틀
- **Input Section**: 
    - 수신 대상 선택용 Radio 버튼 또는 Card 타입 선택기
    - 원문 입력용 Textarea (Placeholder: "내일 보고서 늦을 것 같다고 말해줘")
    - '변환하기' 버튼 (강조 색상)
- **Output Section**: 
    - 변환 결과 표시 영역 (Read-only Textarea 혹은 Div)
    - '결과 복사하기' 버튼

## 8. 데이터 흐름 (Data Flow)
1. 사용자: 수신 대상 선택 및 텍스트 입력 후 버튼 클릭
2. 프론트엔드: `POST /convert` 엔드포인트로 JSON 데이터(target, text) 전송
3. 백엔드(FastAPI): 
    - API 요청 수신
    - LangChain을 통해 Prompt 구성 (대상별 페르소나 설정)
    - Upstage Solar-Pro2 API 호출
4. AI: 변환된 메시지 생성 및 반환
5. 백엔드: 생성된 텍스트를 프론트엔드로 응답
6. 프론트엔드: 결과를 화면에 출력

## 9. 실습 마일스톤 (Milestone)
1. **환경 구축**: Python 환경 설정 및 Upstage API Key 발급
2. **백엔드 개발**: FastAPI 서버 및 LangChain 연동 로직 완성
3. **프론트엔드 개발**: UI 레이아웃 구현 및 API 연동 (Fetch API)
4. **테스트 및 배포**: 상황별 변환 퀄리티 확인 후 Vercel 배포
